## Usage / Component State

### Multiple selection

Pass a `multiple` prop to enable multi-selection.

<ShowcaseMultiple />

### Initial Value (Uncontrolled)

An initial, uncontrolled value can be provided using the `value` prop.

<ShowcaseInitialValue />

If you want to have some control when an item is selected, like making some side effect you can use
the `onChange$`. The event is fired when the user toggle the button, and receives the new value.

<ShowcaseValue />

### Reactive Value (Controlled)

Pass a signal to `bind:value` prop to make the pressed state controlled (binding the value with a signal).

<ShowcaseValueBind />

### Disabled

Pass the `disabled` prop.

<ShowcaseDisabled />

You can also disabled specific items, pass the `disabled` prop at the `ToggleGroup.Item` level:
When navigating with key arrows, the disabled item will be skipped.

<ShowcaseItemDisabledCenter />

### Looping Enabled

Pass the `loop` prop. When enabled, keyboard navigation will loop from last item to first, and vice versa.
If one item is disabled it will skip it.

<ShowcaseLoop />

If one item is disabled it will skip it.

<ShowcaseLoopItemDisabled />

### Vertical Orientation

Pass the `orientation` prop.

<ShowcaseVertical />

### Right-to-Left (rtl) Direction

Pass the `direction` prop to `rtl`.

<ShowcaseHorizontalRtl />
